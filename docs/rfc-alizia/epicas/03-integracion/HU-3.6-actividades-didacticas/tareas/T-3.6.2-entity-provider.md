# T-3.6.2: Entity y Provider de Activity

**Historia:** HU-3.6 — Actividades didacticas
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripcion

Definir la entity `Activity` en Go con tags de GORM mapeando el enum `class_moment` como string. Crear la interface `ActivityProvider` con los metodos para crear y listar actividades con filtro opcional por momento.

## Implementacion

### `src/core/entities/activity.go`

```go
package entities

import "time"

// ClassMoment representa los momentos de una clase
type ClassMoment string

const (
    MomentApertura   ClassMoment = "apertura"
    MomentDesarrollo ClassMoment = "desarrollo"
    MomentCierre     ClassMoment = "cierre"
)

type Activity struct {
    ID              uint        `json:"id" gorm:"primaryKey"`
    OrganizationID  uint        `json:"organization_id" gorm:"not null"`
    Moment          ClassMoment `json:"moment" gorm:"type:class_moment;not null"`
    Name            string      `json:"name" gorm:"type:varchar(255);not null"`
    Description     *string     `json:"description,omitempty" gorm:"type:text"`
    DurationMinutes *int        `json:"duration_minutes,omitempty"`
    CreatedAt       time.Time   `json:"created_at" gorm:"autoCreateTime"`
}

func (Activity) TableName() string {
    return "activities"
}

// ValidMoments retorna los momentos validos para validacion
func ValidMoments() []ClassMoment {
    return []ClassMoment{MomentApertura, MomentDesarrollo, MomentCierre}
}
```

### `src/core/providers/activity.go`

```go
package providers

import (
    "context"

    "github.com/educabot/demo-alizia/src/core/entities"
)

type ActivityProvider interface {
    // Create crea una nueva actividad didactica
    Create(ctx context.Context, activity *entities.Activity) (*entities.Activity, error)

    // List retorna todas las actividades de una organizacion, con filtro opcional por momento
    List(ctx context.Context, orgID uint, moment *entities.ClassMoment) ([]entities.Activity, error)

    // ListByMoment retorna actividades de una org filtradas por momento especifico
    ListByMoment(ctx context.Context, orgID uint, moment entities.ClassMoment) ([]entities.Activity, error)
}
```

## Notas

- `ClassMoment` se define como `type string` para aprovechar el type system de Go. Las constantes aseguran que solo se usen valores validos desde el codigo.
- GORM mapea `ClassMoment` al enum de PostgreSQL con `gorm:"type:class_moment"`. Al insertar, el valor string se castea automaticamente.
- `Description` y `DurationMinutes` son punteros para representar NULL en la base de datos y omitirse del JSON cuando no tienen valor.
- `List` recibe `moment` como puntero para hacerlo opcional: si es nil, retorna todas las actividades; si tiene valor, filtra por ese momento.
- `ListByMoment` es un atajo de conveniencia para cuando se sabe con certeza que se quiere filtrar.
