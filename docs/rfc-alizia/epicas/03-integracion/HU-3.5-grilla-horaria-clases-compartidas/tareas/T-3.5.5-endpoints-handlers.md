# T-3.5.5: Endpoints y handlers

**Historia:** HU-3.5 — Grilla horaria y clases compartidas
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripcion

Implementar los endpoints HTTP para crear time slots, obtener la grilla horaria de un curso y consultar los numeros de clases compartidas para un course_subject. Incluye validacion de la configuracion `shared_classes_enabled` de la organizacion.

## Implementacion

### Endpoints

| Metodo | Ruta | Descripcion | Rol |
|--------|------|-------------|-----|
| POST | `/api/v1/courses/:id/time-slots` | Crear time slot con subjects | admin |
| GET | `/api/v1/courses/:id/time-slots` | Listar grilla horaria del curso | auth |
| GET | `/api/v1/course-subjects/:id/shared-class-numbers` | Numeros de clases compartidas | auth |

### Request body para POST

```json
{
    "day_of_week": 1,
    "start_time": "08:00",
    "end_time": "09:30",
    "course_subject_ids": [1, 5]
}
```

### Handler: CreateTimeSlot

```go
func (h *TimeSlotHandler) Create(c *gin.Context) {
    courseID := parseUintParam(c, "id")

    var req struct {
        DayOfWeek        int    `json:"day_of_week" binding:"required,min=0,max=6"`
        StartTime        string `json:"start_time" binding:"required"`
        EndTime          string `json:"end_time" binding:"required"`
        CourseSubjectIDs []uint `json:"course_subject_ids" binding:"required,min=1,max=2"`
    }
    if err := c.ShouldBindJSON(&req); err != nil {
        c.JSON(400, gin.H{"error": err.Error()})
        return
    }

    // Validar shared_classes_enabled si hay 2 subjects
    if len(req.CourseSubjectIDs) > 1 {
        org, _ := h.orgProvider.GetByID(c, currentOrgID(c))
        config := parseConfig(org.Config)
        if !config.SharedClassesEnabled {
            c.JSON(422, gin.H{"error": "shared classes are not enabled for this organization"})
            return
        }
    }

    // Construir entity con subjects
    slot := &entities.TimeSlot{
        CourseID:  courseID,
        DayOfWeek: req.DayOfWeek,
        StartTime: req.StartTime,
        EndTime:   req.EndTime,
    }
    for _, csID := range req.CourseSubjectIDs {
        slot.Subjects = append(slot.Subjects, entities.TimeSlotSubject{
            CourseSubjectID: csID,
        })
    }

    result, err := h.provider.Create(c, slot)
    if err != nil {
        // El trigger puede lanzar error si course_subject no pertenece al curso
        c.JSON(422, gin.H{"error": err.Error()})
        return
    }
    c.JSON(201, result)
}
```

### Handler: GetSharedClassNumbers

```go
func (h *TimeSlotHandler) GetSharedClassNumbers(c *gin.Context) {
    csID := parseUintParam(c, "id")
    totalClasses, _ := strconv.Atoi(c.DefaultQuery("total_classes", "0"))

    if totalClasses <= 0 {
        c.JSON(400, gin.H{"error": "total_classes query param is required and must be > 0"})
        return
    }

    numbers, err := h.provider.GetSharedClassNumbers(c, csID, totalClasses)
    if err != nil {
        c.JSON(500, gin.H{"error": err.Error()})
        return
    }
    c.JSON(200, gin.H{"shared_class_numbers": numbers})
}
```

### Registro en `mapping.go`

```go
// Time Slots
courses.POST("/:id/time-slots", timeSlotHandler.Create)
courses.GET("/:id/time-slots", timeSlotHandler.ListByCourse)

// Shared class numbers
courseSubjects.GET("/:id/shared-class-numbers", timeSlotHandler.GetSharedClassNumbers)
```

## Notas

- La validacion de `shared_classes_enabled` se hace en el handler antes de llamar al provider. Si esta deshabilitada y se envian 2 subjects, se retorna 422.
- El trigger de DB (T-3.5.2) es la segunda linea de defensa: si un course_subject no pertenece al curso, PostgreSQL rechaza la insercion.
- El endpoint GET de time-slots enriquece la respuesta del detalle de curso (HU-3.4) con la grilla horaria completa.
- `total_classes` es un query param obligatorio para `shared-class-numbers` porque el backend no conoce la duracion del periodo sin contexto adicional.
