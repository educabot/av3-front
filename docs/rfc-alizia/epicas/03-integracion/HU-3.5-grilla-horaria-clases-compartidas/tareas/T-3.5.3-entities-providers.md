# T-3.5.3: Entities y Providers de TimeSlot

**Historia:** HU-3.5 — Grilla horaria y clases compartidas
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripcion

Definir las entities `TimeSlot` y `TimeSlotSubject` en Go con tags de GORM, incluyendo relaciones de preload. Crear la interface `TimeSlotProvider` con los metodos necesarios para gestionar la grilla horaria y consultar clases compartidas.

## Implementacion

### `src/core/entities/time_slot.go`

```go
package entities

import "time"

type TimeSlot struct {
    ID          uint              `json:"id" gorm:"primaryKey"`
    CourseID    uint              `json:"course_id" gorm:"not null"`
    DayOfWeek   int               `json:"day_of_week" gorm:"type:smallint;not null"`
    StartTime   string            `json:"start_time" gorm:"type:time;not null"`
    EndTime     string            `json:"end_time" gorm:"type:time;not null"`
    CreatedAt   time.Time         `json:"created_at" gorm:"autoCreateTime"`
    Subjects    []TimeSlotSubject `json:"subjects" gorm:"foreignKey:TimeSlotID"`
}

func (TimeSlot) TableName() string {
    return "time_slots"
}

type TimeSlotSubject struct {
    ID              uint           `json:"id" gorm:"primaryKey"`
    TimeSlotID      uint           `json:"time_slot_id" gorm:"not null"`
    CourseSubjectID uint           `json:"course_subject_id" gorm:"not null"`
    CourseSubject   *CourseSubject `json:"course_subject,omitempty" gorm:"foreignKey:CourseSubjectID"`
}

func (TimeSlotSubject) TableName() string {
    return "time_slot_subjects"
}
```

### `src/core/providers/time_slot.go`

```go
package providers

import (
    "context"

    "github.com/educabot/demo-alizia/src/core/entities"
)

// SharedClassInfo representa un slot compartido detectado
type SharedClassInfo struct {
    DayOfWeek        int    `json:"day_of_week"`
    StartTime        string `json:"start_time"`
    EndTime          string `json:"end_time"`
    CourseSubjectIDs []uint `json:"course_subject_ids"`
}

type TimeSlotProvider interface {
    // Create crea un time_slot junto con sus subjects asociados
    Create(ctx context.Context, slot *entities.TimeSlot) (*entities.TimeSlot, error)

    // ListByCourse retorna todos los time_slots de un curso con subjects precargados
    ListByCourse(ctx context.Context, courseID uint) ([]entities.TimeSlot, error)

    // GetSharedClasses detecta slots compartidos (2+ subjects) para un curso
    GetSharedClasses(ctx context.Context, courseID uint) ([]SharedClassInfo, error)

    // GetSharedClassNumbers calcula que class_numbers son compartidos para un course_subject
    GetSharedClassNumbers(ctx context.Context, courseSubjectID uint, totalClasses int) ([]int, error)
}
```

## Notas

- `TimeSlot.Subjects` usa `foreignKey:TimeSlotID` para que GORM haga el preload automaticamente con `.Preload("Subjects")`.
- `TimeSlotSubject.CourseSubject` es un puntero con `omitempty` para que sea opcional en el JSON (solo se incluye cuando se hace preload).
- `StartTime` y `EndTime` se mapean como `string` en Go (formato "HH:MM:SS") ya que GORM no tiene un tipo nativo para TIME de PostgreSQL.
- `GetSharedClassNumbers` recibe `totalClasses` para poder calcular la proyeccion semanal a lo largo de todo el periodo.
- La interface sera implementada por el repository en T-3.5.4.
