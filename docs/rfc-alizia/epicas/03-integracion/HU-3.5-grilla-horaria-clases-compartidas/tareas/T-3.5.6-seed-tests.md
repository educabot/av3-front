## Descripcion

Crear datos semilla con grillas horarias para 2-3 cursos incluyendo clases normales y compartidas. Escribir tests que cubran la creacion de slots, validacion del trigger, rechazo por configuracion y calculo de numeros de clases compartidas.

## Notas

- Los IDs del seed son referenciales. Ajustar segun los datos reales generados por seeds anteriores (HU-3.4).
- Se usa `currval('time_slots_id_seq')` para vincular el slot recien creado con sus subjects sin necesidad de conocer el ID de antemano.
- El test del trigger verifica la proteccion a nivel de base de datos, independiente de la validacion del handler.
- El calculo de shared class numbers debe probarse con distintas cantidades de `total_classes` para verificar la proyeccion semanal.
