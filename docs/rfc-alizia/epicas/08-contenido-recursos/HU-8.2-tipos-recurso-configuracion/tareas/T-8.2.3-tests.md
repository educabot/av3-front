# T-8.2.3: Tests

**Historia:** HU-8.2 — Tipos de recurso y configuración por organización
**Tipo:** Testing
**Estado:** ⬜ Pendiente

---

## Descripción

Tests para endpoints de tipos de recurso y lógica de resolución de prompt/schema.

## Casos de test

### Endpoints

| Test | Input | Expected |
|------|-------|----------|
| Listar tipos sin config | Org sin registros en org_resource_types | Todos los públicos disponibles |
| Listar con tipo deshabilitado | Org con enabled=false para lecture_guide | lecture_guide no aparece |
| Listar con tipo privado | Org con tipo custom | Públicos + privado de la org |
| Tipo privado de otra org | Org B intenta ver tipo de Org A | No aparece |
| Configurar tipo | PUT con custom_prompt | UPSERT correcto |

### Resolución de prompt/schema

| Test | Input | Expected |
|------|-------|----------|
| Sin config de org | orgConfig nil | Usa defaults del tipo |
| Custom prompt | orgConfig con custom_prompt | Usa custom_prompt |
| Custom schema | orgConfig con custom_output_schema | Usa custom schema |
| Custom prompt vacío | orgConfig con custom_prompt = "" | Usa default (ignora vacío) |
| Ambos custom | custom_prompt + custom_schema | Usa ambos custom |

## Notas

- Tests de resolución son puramente unitarios (no DB)
- Tests de endpoints verifican el filtrado correcto por org
