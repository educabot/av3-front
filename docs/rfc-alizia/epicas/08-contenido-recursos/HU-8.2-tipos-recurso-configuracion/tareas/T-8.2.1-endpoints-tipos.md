# T-8.2.1: Endpoints de tipos de recurso

**Historia:** HU-8.2 — Tipos de recurso y configuración por organización
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Implementar los endpoints para consultar y configurar tipos de recurso por organización.

## Implementación

### Endpoints

| Método | Ruta | Descripción |
|--------|------|-------------|
| GET | `/resource-types` | Tipos disponibles para la org del usuario |
| GET | `/resource-types/{id}` | Detalle de un tipo con prompt/schema resuelto |
| PUT | `/resource-types/{id}/config` | Configurar tipo para la org (habilitar, custom prompt) |

### Listar tipos disponibles

```
GET /resource-types
```

Respuesta:
```json
[
    {
        "id": 1,
        "slug": "lecture_guide",
        "name": "Guía de lectura",
        "description": "Guía estructurada para acompañar la lectura...",
        "requires_font": true,
        "is_customized": false
    },
    {
        "id": 2,
        "slug": "course_sheet",
        "name": "Ficha de cátedra",
        "description": "Documento de referencia sobre un tema...",
        "requires_font": false,
        "is_customized": true
    },
    {
        "id": 3,
        "slug": "free_creation",
        "name": "Creación libre",
        "description": "Recurso libre donde el docente define los lineamientos",
        "requires_font": false,
        "is_customized": false
    }
]
```

Lógica:
1. Obtener tipos públicos (organization_id NULL)
2. Filtrar por habilitados en `organization_resource_types` (si no hay registro, se asume habilitado)
3. Agregar tipos privados de la org
4. Marcar `is_customized: true` si hay custom_prompt o custom_output_schema

### Configurar tipo

```
PUT /resource-types/{id}/config
```

```json
{
    "enabled": true,
    "custom_prompt": "Generá una guía de lectura para alumnos de nivel secundario...",
    "custom_output_schema": null
}
```

Hace UPSERT en `organization_resource_types`.

## Notas

- La lista filtra automáticamente por la org del usuario autenticado
- Si un tipo público no tiene registro en `organization_resource_types`, se asume habilitado por defecto
- El endpoint de configuración es para admins/coordinadores, no para docentes
- `is_customized` es un campo calculado que indica si la org sobreescribió algo
