# T-8.3.2: Endpoint de creación y generación

**Historia:** HU-8.3 — Generación de recursos con IA
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Implementar el endpoint REST para crear y generar un recurso en un solo paso.

## Implementación

### Endpoint

```
POST /resources/generate
```

Request:
```json
{
    "resource_type_id": 1,
    "course_subject_id": 5,
    "title": "Guía de lectura - Martín Fierro",
    "font_reference": "Martín Fierro, José Hernández, Capítulos 1-3",
    "user_instructions": null
}
```

Response (201):
```json
{
    "id": 42,
    "resource_type": {
        "id": 1,
        "slug": "lecture_guide",
        "name": "Guía de lectura"
    },
    "course_subject": {
        "id": 5,
        "subject": "Lengua y Literatura",
        "course": "3a"
    },
    "title": "Guía de lectura - Martín Fierro",
    "content": {
        "introduction": "Esta guía acompaña la lectura de los primeros tres capítulos...",
        "pre_reading": [
            "¿Qué sabés sobre la vida en el campo argentino en el siglo XIX?"
        ],
        "during_reading": [
            {
                "question": "¿Cómo describe Hernández la vida del gaucho?",
                "page_reference": "Cap. 1, estrofas 1-10"
            }
        ],
        "post_reading": [
            "Escribí un párrafo comparando la vida del gaucho con la vida actual"
        ],
        "synthesis": "Los primeros capítulos presentan..."
    },
    "font_reference": "Martín Fierro, José Hernández, Capítulos 1-3",
    "status": "generated",
    "created_by": {
        "id": 3,
        "name": "Doc. García"
    },
    "created_at": "2026-03-27T14:00:00Z"
}
```

### Creación libre

```
POST /resources/generate
```

```json
{
    "resource_type_id": 3,
    "course_subject_id": 5,
    "title": "Actividad de comprensión lectora",
    "font_reference": null,
    "user_instructions": "Quiero una actividad de comprensión lectora para alumnos de 3er año sobre un cuento corto de Borges. Que tenga preguntas de nivel literal, inferencial y crítico."
}
```

### Validaciones

| Validación | Condición | Error |
|-----------|-----------|-------|
| Tipo existe | resource_type_id válido | 404 |
| Tipo habilitado | No deshabilitado para la org | 403 |
| Fuente requerida | requires_font && font_reference vacío | 400 |
| Título requerido | title vacío | 400 |

## Notas

- El endpoint combina creación + generación en un solo paso para simplificar el flujo
- `organization_id` y `created_by` se resuelven del token del usuario autenticado
- `course_subject_id` es opcional — el recurso puede no tener contexto de disciplina
- La respuesta incluye el content completo para que el frontend lo renderice inmediatamente
