# T-8.3.3: Context builder para recursos

**Historia:** HU-8.3 — Generación de recursos con IA
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Implementar el context builder que enriquece el prompt de generación de recursos con datos del dominio: disciplina, curso, alumnos, temas curriculares y fuente seleccionada.

## Implementación

```go
// src/core/usecases/ai/resource_context_builder.go
package ai

// BuildResourceContext construye el contexto para el user prompt de generación de recursos.
func BuildResourceContext(req GenerateResourceRequest, resourceType *entities.ResourceType) string {
    var parts []string

    parts = append(parts, fmt.Sprintf("Tipo de recurso: %s", resourceType.Name))
    parts = append(parts, fmt.Sprintf("Título: %s", req.Title))

    if req.FontReference != nil {
        parts = append(parts, fmt.Sprintf("Fuente: %s", *req.FontReference))
    }

    return strings.Join(parts, "\n")
}

// BuildResourceContextEnriched incluye datos del course_subject si está disponible.
func BuildResourceContextEnriched(
    req GenerateResourceRequest,
    resourceType *entities.ResourceType,
    cs *entities.CourseSubject,
    topics []entities.Topic,
) string {
    base := BuildResourceContext(req, resourceType)

    var enriched []string
    enriched = append(enriched, base)

    if cs != nil {
        enriched = append(enriched, fmt.Sprintf("\nContexto del aula:"))
        enriched = append(enriched, fmt.Sprintf("- Disciplina: %s", cs.Subject.Name))
        enriched = append(enriched, fmt.Sprintf("- Curso: %s", cs.Course.Name))
        enriched = append(enriched, fmt.Sprintf("- Docente: %s", cs.Teacher.Name))

        if cs.Course.StudentCount > 0 {
            enriched = append(enriched, fmt.Sprintf("- Alumnos: %d", cs.Course.StudentCount))
        }
    }

    if len(topics) > 0 {
        enriched = append(enriched, "\nTemas curriculares relacionados:")
        for _, t := range topics {
            enriched = append(enriched, fmt.Sprintf("- %s", t.Name))
        }
    }

    return strings.Join(enriched, "\n")
}
```

### Ejemplo de prompt completo enviado al LLM

```
System prompt (del resource_type o custom de la org):
"Generá una guía de lectura para alumnos de nivel secundario..."

User prompt (contexto + fuente):
"Tipo de recurso: Guía de lectura
Título: Guía de lectura - Martín Fierro
Fuente: Martín Fierro, José Hernández, Capítulos 1-3

Contexto del aula:
- Disciplina: Lengua y Literatura
- Curso: 3a
- Docente: Doc. García
- Alumnos: 28

Temas curriculares relacionados:
- Literatura argentina del siglo XIX
- Poesía gauchesca"
```

## Notas

- El contexto enriquecido permite que Alizia adapte el recurso al nivel del curso y la disciplina
- Si no hay course_subject, se usa el contexto básico (tipo + título + fuente)
- Los topics se obtienen del coordination_document vigente si existe uno publicado para la disciplina
- El context builder es una función pura — fácil de testear sin mocks
- El count de alumnos permite que Alizia sugiera actividades grupales adecuadas al tamaño del curso
