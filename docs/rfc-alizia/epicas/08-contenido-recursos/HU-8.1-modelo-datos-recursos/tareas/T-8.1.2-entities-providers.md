# T-8.1.2: Entities y providers

**Historia:** HU-8.1 — Modelo de datos de recursos
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Definir las entities Go con GORM tags y las interfaces de provider para CRUD de fonts, recursos y tipos de recurso.

## Implementación

### Entities

```go
// src/core/entities/font.go
package entities

import "time"

type Font struct {
    ID             int       `json:"id" gorm:"primaryKey"`
    OrganizationID int       `json:"organization_id"`
    Name           string    `json:"name"`
    Description    *string   `json:"description,omitempty"`
    FileURL        string    `json:"file_url"`
    FileType       *string   `json:"file_type,omitempty"`
    ThumbnailURL   *string   `json:"thumbnail_url,omitempty"`
    AreaID         *int      `json:"area_id,omitempty"`
    IsValidated    bool      `json:"is_validated"`
    CreatedAt      time.Time `json:"created_at"`

    Area *Area `json:"area,omitempty" gorm:"foreignKey:AreaID"`
}

// src/core/entities/resource_type.go
package entities

import "time"

type ResourceType struct {
    ID             int       `json:"id" gorm:"primaryKey"`
    Slug           string    `json:"slug"`
    Name           string    `json:"name"`
    Description    *string   `json:"description,omitempty"`
    Prompt         string    `json:"prompt"`
    OutputSchema   JSON      `json:"output_schema" gorm:"type:jsonb"`
    RequiresFont   bool      `json:"requires_font"`
    IsActive       bool      `json:"is_active"`
    OrganizationID *int      `json:"organization_id,omitempty"`
    CreatedAt      time.Time `json:"created_at"`
}

// src/core/entities/organization_resource_type.go
type OrganizationResourceType struct {
    ID               int       `json:"id" gorm:"primaryKey"`
    OrganizationID   int       `json:"organization_id"`
    ResourceTypeID   int       `json:"resource_type_id"`
    Enabled          bool      `json:"enabled"`
    CustomPrompt     *string   `json:"custom_prompt,omitempty"`
    CustomOutputSchema *JSON   `json:"custom_output_schema,omitempty" gorm:"type:jsonb"`
    CreatedAt        time.Time `json:"created_at"`

    ResourceType ResourceType `json:"resource_type" gorm:"foreignKey:ResourceTypeID"`
}

// src/core/entities/resource.go
type ResourceStatus string

const (
    ResourceDraft     ResourceStatus = "draft"
    ResourceGenerated ResourceStatus = "generated"
    ResourcePublished ResourceStatus = "published"
)

type Resource struct {
    ID              int            `json:"id" gorm:"primaryKey"`
    OrganizationID  int            `json:"organization_id"`
    ResourceTypeID  int            `json:"resource_type_id"`
    CourseSubjectID *int           `json:"course_subject_id,omitempty"`
    Title           string         `json:"title"`
    Content         JSON           `json:"content" gorm:"type:jsonb;default:'{}'"`
    FontID          *int           `json:"font_id,omitempty"`
    Status          ResourceStatus `json:"status"`
    CreatedBy       int            `json:"created_by"`
    CreatedAt       time.Time      `json:"created_at"`
    UpdatedAt       time.Time      `json:"updated_at"`

    ResourceType  ResourceType  `json:"resource_type" gorm:"foreignKey:ResourceTypeID"`
    CourseSubject *CourseSubject `json:"course_subject,omitempty" gorm:"foreignKey:CourseSubjectID"`
    Font          *Font         `json:"font,omitempty" gorm:"foreignKey:FontID"`
    Creator       User          `json:"creator" gorm:"foreignKey:CreatedBy"`
}
```

### Providers

```go
// src/core/providers/resource_provider.go
package providers

type FontProvider interface {
    GetByOrgID(orgID int) ([]entities.Font, error)
    GetByID(id int) (*entities.Font, error)
    Create(f *entities.Font) error
    Update(f *entities.Font) error
    Delete(id int) error
}

type ResourceTypeProvider interface {
    GetAll() ([]entities.ResourceType, error)
    GetByID(id int) (*entities.ResourceType, error)
    GetAvailableForOrg(orgID int) ([]entities.ResourceType, error) // públicos + privados de la org
}

type OrganizationResourceTypeProvider interface {
    GetByOrgID(orgID int) ([]entities.OrganizationResourceType, error)
    GetByOrgAndType(orgID, typeID int) (*entities.OrganizationResourceType, error)
    Upsert(ort *entities.OrganizationResourceType) error
}

type ResourceProvider interface {
    Create(r *entities.Resource) error
    GetByID(id int) (*entities.Resource, error)
    GetByOrgID(orgID int, filters ResourceFilters) ([]entities.Resource, error)
    Update(r *entities.Resource) error
    Delete(id int) error
}

type ResourceFilters struct {
    ResourceTypeID  *int
    CourseSubjectID *int
    SubjectID       *int    // filtro soft por disciplina
    FontID          *int
    Status          *string
    CreatedBy       *int
    Limit           int
    Offset          int
}
```

## Notas

- `GetAvailableForOrg` retorna tipos donde `organization_id IS NULL OR organization_id = orgID`
- `ResourceFilters.SubjectID` filtra via join con course_subjects — es un filtro UX, no de permisos
- El `Content` es JSONB cuya estructura varía según el `output_schema` del tipo de recurso
- `Creator` se preloadea para mostrar "Creado por Doc. García" en la biblioteca
- `Font` se preloadea cuando `font_id` no es null
