### Edición inline

```javascript
// Cada sección del content es editable
function renderEditableSection(key, value, schema) {
    const section = document.createElement('div');
    section.className = 'resource-section';

    // Header con nombre legible + botón editar
    const header = renderSectionHeader(key, schema);
    section.appendChild(header);

    // Contenido: string → textarea; array → lista con add/remove; object → campos
    if (typeof value === 'string') {
        section.appendChild(renderEditableText(key, value));
    } else if (Array.isArray(value)) {
        section.appendChild(renderEditableList(key, value, schema.items));
    }

    return section;
}

// Guardar cambios al salir del modo edición
async function saveSection(key, newValue) {
    const content = { ...state.currentResource.content };
    content[key] = newValue;
    await fetch(`/resources/${state.currentResource.id}`, {
        method: 'PATCH',
        body: JSON.stringify({ content })
    });
}
```

### Exportación para impresión

```javascript
function exportForPrint(resource) {
    // Genera una ventana/iframe con estilos de impresión
    const printWindow = window.open('', '_blank');
    printWindow.document.write(`
        <html>
        <head>
            <title>${resource.title}</title>
            <style>
                body { font-family: serif; max-width: 700px; margin: 0 auto; padding: 40px; }
                h1 { font-size: 20px; border-bottom: 2px solid #333; }
                h2 { font-size: 16px; margin-top: 24px; }
                ul { padding-left: 20px; }
                .footer { margin-top: 40px; font-size: 12px; color: #666; }
            </style>
        </head>
        <body>
            ${renderContentForPrint(resource)}
            <div class="footer">
                ${resource.resource_type.name} · ${resource.course_subject?.subject || ''} · ${resource.creator.name}
            </div>
        </body>
        </html>
    `);
    printWindow.print();
}
```

## Notas

- La edición inline usa double-click para entrar en modo edición (consistente con doc de coordinación)
- Las secciones de tipo array tienen botón "+" para agregar y "×" para eliminar items
- El chat con Alizia se muestra debajo del contenido (no en panel lateral, para aprovechar ancho)
- La exportación genera una vista limpia para impresión con estilos serif y sin chrome de la app
- Incluye footer con tipo, disciplina y creador para contexto en la impresión
- El botón "Publicar" cambia el estado y muestra badge verde
