# T-8.5.4: Tests

**Historia:** HU-8.5 — Edición y exportación
**Tipo:** Testing
**Estado:** ⬜ Pendiente

---

## Descripción

Tests para edición de recursos, chat con Alizia y publicación.

## Casos de test

### Edición

| Test | Input | Expected |
|------|-------|----------|
| PATCH content válido | Content completo actualizado | 200, content guardado |
| PATCH solo título | Solo title en body | 200, título actualizado, content intacto |
| PATCH como no-creador | Otro docente de la org | 403 Forbidden |
| PATCH recurso de otra org | Resource de org B | 404 |
| DELETE recurso propio | Creador elimina | 200, recurso eliminado |
| DELETE como no-creador | Otro docente elimina | 403 |

### Publicación

| Test | Input | Expected |
|------|-------|----------|
| Publicar recurso generado | POST publish, status=generated | 200, status=published |
| Publicar recurso draft | POST publish, status=draft | 200, status=published |
| Publicar ya publicado | POST publish, status=published | 200, no-op (ya está publicado) |

### Chat con Alizia

| Test | Input | Expected |
|------|-------|----------|
| Editar sección via chat | "Cambiá la introducción" | Tool call update_resource_section ejecutado |
| Agregar item via chat | "Agregá una pregunta" | Tool call append_to_section ejecutado |
| Eliminar item via chat | "Sacá la segunda pregunta" | Tool call remove_from_section ejecutado |
| Chat sin tools | "¿Qué te parece el recurso?" | Respuesta sin tool calls |
| Historial preservado | 3 turnos de conversación | Contexto mantenido |

### Seguridad

| Test | Input | Expected |
|------|-------|----------|
| Chat como no-creador | Otro docente chatea | 403 |
| Chat recurso otra org | Recurso de org B | 404 |

## Notas

- Tests de chat requieren mock del AI Provider
- Verificar que los tool calls actualizan correctamente el content JSONB
- Verificar que append_to_section agrega al final del array
- Verificar que remove_from_section con índice inválido retorna error
