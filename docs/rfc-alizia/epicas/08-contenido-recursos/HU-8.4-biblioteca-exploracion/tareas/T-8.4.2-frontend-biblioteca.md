### Renderizado dinámico de content

```javascript
// Renderiza el JSONB content según el output_schema del tipo
function renderResourceContent(content, schema) {
    const container = document.createElement('div');

    for (const [key, schemaDef] of Object.entries(schema.properties)) {
        const value = content[key];
        if (!value) continue;

        if (schemaDef.type === 'string') {
            // Renderizar como párrafo
            container.appendChild(renderTextField(key, value));
        } else if (schemaDef.type === 'array') {
            // Renderizar como lista
            container.appendChild(renderArrayField(key, value, schemaDef.items));
        } else if (schemaDef.type === 'object') {
            // Renderizar como bloque
            container.appendChild(renderObjectField(key, value));
        }
    }

    return container;
}
```

## Notas

- El renderizado dinámico permite que diferentes tipos de recurso se muestren correctamente sin templates hardcodeados
- Los filtros se aplican como query params al endpoint GET /resources
- La búsqueda tiene debounce de 300ms para no spamear el backend
- El checkbox "Mis recursos" filtra por `created_by` del usuario actual
- Paginación simple con offset (suficiente para el volumen esperado)
