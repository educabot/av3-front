# T-1.1.4: Config: JWT auth env vars

**Historia:** HU-1.1 — Autenticación JWT
**Tipo:** Backend
**Estado:** ✅ Completada

---

## Descripción

Agregar las variables de configuración de JWT auth al config struct de la app.

## Implementación

```go
// config/config.go
type Config struct {
    config.BaseConfig // de team-ai-toolkit — ya incluye JWTSecret
}

func Load() Config {
    base := config.LoadBase()
    return Config{
        BaseConfig: base,
    }
}
```

`JWTSecret` ya viene incluido en `BaseConfig` de team-ai-toolkit, no hace falta agregarlo manualmente.

## Variables de entorno

| Variable | Descripción | Ejemplo |
|----------|-------------|---------|
| `JWT_SECRET` | Secret for JWT token validation | `<jwt-secret-staging>` |

## Notas

- `MustEnv` panics si la variable no existe — fail fast al startup
- `BaseConfig` ya trae: `PORT`, `DATABASE_URL`, `ENV`, `JWTSecret`, `BUGSNAG_API_KEY`
