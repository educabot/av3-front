# T-5.1.1: Migracion — tablas de planificacion

**Historia:** HU-5.1 — Modelo de datos planificacion
**Tipo:** DB
**Estado:** ⬜ Pendiente

---

## Descripcion

Crear las tablas `teacher_lesson_plans`, `lesson_plan_activities`, `lesson_plan_topics` y `lesson_plan_moment_fonts` con sus enums y constraints.

## Implementacion

```sql
-- Enum para estado de planificacion
CREATE TYPE lesson_plan_status AS ENUM ('pending', 'in_progress', 'published');

-- Enum para tipo de fuente (nullable)
CREATE TYPE source_type AS ENUM ('resource', 'custom');

-- Enum para modo de asignación de fonts
CREATE TYPE resources_mode AS ENUM ('global', 'per_moment');

CREATE TABLE teacher_lesson_plans (
    id SERIAL PRIMARY KEY,
    organization_id INTEGER NOT NULL REFERENCES organizations(id),
    course_subject_id INTEGER NOT NULL REFERENCES course_subjects(id),
    coordination_document_id INTEGER NOT NULL REFERENCES coordination_documents(id),
    coord_doc_class_id INTEGER NOT NULL REFERENCES coord_doc_classes(id),
    class_number INTEGER NOT NULL,
    title VARCHAR(500) NOT NULL,
    objective TEXT,
    knowledge_content TEXT,
    didactic_strategies TEXT,
    class_format VARCHAR(100),
    custom_instruction TEXT,
    resources_mode resources_mode,
    status lesson_plan_status NOT NULL DEFAULT 'pending',
    proposal TEXT,
    source_type source_type,
    source_reference TEXT,
    created_at TIMESTAMP NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMP NOT NULL DEFAULT NOW(),
    UNIQUE(course_subject_id, coord_doc_class_id)
);

CREATE TABLE lesson_plan_activities (
    id SERIAL PRIMARY KEY,
    teacher_lesson_plan_id INTEGER NOT NULL REFERENCES teacher_lesson_plans(id) ON DELETE CASCADE,
    moment class_moment NOT NULL,
    activity_id INTEGER NOT NULL REFERENCES activities(id),
    custom_note TEXT,
    "order" INTEGER NOT NULL DEFAULT 0,
    UNIQUE(teacher_lesson_plan_id, moment, activity_id)
);

CREATE INDEX idx_lesson_plans_course_subject ON teacher_lesson_plans(course_subject_id);
CREATE INDEX idx_lesson_plans_coord_doc ON teacher_lesson_plans(coordination_document_id);
CREATE INDEX idx_lesson_plan_activities_plan ON lesson_plan_activities(teacher_lesson_plan_id);

-- Junction: topics cubiertos en un lesson plan
CREATE TABLE lesson_plan_topics (
    id SERIAL PRIMARY KEY,
    lesson_plan_id INTEGER NOT NULL REFERENCES teacher_lesson_plans(id) ON DELETE CASCADE,
    topic_id INTEGER NOT NULL REFERENCES topics(id),
    UNIQUE(lesson_plan_id, topic_id)
);

-- Junction: fonts por momento o global en un lesson plan
CREATE TABLE lesson_plan_moment_fonts (
    id SERIAL PRIMARY KEY,
    lesson_plan_id INTEGER NOT NULL REFERENCES teacher_lesson_plans(id) ON DELETE CASCADE,
    moment class_moment,  -- NULL = global (aplica a todos los momentos)
    font_id INTEGER NOT NULL REFERENCES fonts(id),
    UNIQUE(lesson_plan_id, moment, font_id)
);

CREATE INDEX idx_lesson_plan_topics_plan ON lesson_plan_topics(lesson_plan_id);
CREATE INDEX idx_lesson_plan_topics_topic ON lesson_plan_topics(topic_id);
CREATE INDEX idx_lesson_plan_moment_fonts_plan ON lesson_plan_moment_fonts(lesson_plan_id);
CREATE INDEX idx_lesson_plan_moment_fonts_font ON lesson_plan_moment_fonts(font_id);
```

## Notas

- Reutiliza el enum `class_moment` creado en HU-3.6 (opening, development, closing)
- `source_type` es nullable: null = sin fuente anclada
- `proposal` es nullable: se llena al generar con IA
- ON DELETE CASCADE en activities para que al borrar un plan se borren sus actividades
- `lesson_plan_topics` traza qué topics cubre cada lesson plan — permite traceability desde coordinación
- `lesson_plan_moment_fonts` permite asignar fonts por momento (apertura/desarrollo/cierre) o globalmente (moment = NULL)
- `lesson_plan_moment_fonts` depende de la tabla `fonts` (Épica 8, HU-8.1)
- `resources_mode` indica si las fonts se asignan globalmente o por momento
- `knowledge_content`, `didactic_strategies` y `class_format` capturan el contenido pedagógico del plan
- `custom_instruction` es texto libre del docente para guiar la generación IA
