# T-5.6.4: Endpoints CRUD de bitácora

**Historia:** HU-5.6 — Bitácora de cotejo
**Tipo:** Backend
**Estado:** ⬜ Pendiente
**Fase:** Post-MVP

---

## Descripción

Implementar los endpoints REST para crear, leer y actualizar bitácoras de cotejo. Incluye endpoints para el docente (escritura) y para el coordinador (lectura de su equipo).

## Implementación

### Endpoints

| Método | Ruta | Descripción | Rol |
|--------|------|-------------|-----|
| POST | `/lesson-plans/{id}/log-entry` | Crear bitácora (texto) | Docente |
| POST | `/lesson-plans/{id}/log-entry/audio` | Crear bitácora (audio) | Docente |
| GET | `/lesson-plans/{id}/log-entry` | Obtener bitácora del plan | Docente / Coordinador |
| PATCH | `/lesson-plans/{id}/log-entry` | Actualizar datos manualmente | Docente |
| POST | `/lesson-plans/{id}/log-entry/chat` | Chat de seguimiento con Alizia | Docente |
| GET | `/course-subjects/{id}/log-entries` | Listar bitácoras de una disciplina | Coordinador |
| GET | `/areas/{id}/log-entries` | Listar bitácoras del área | Coordinador |

### Crear bitácora (texto)

```
POST /lesson-plans/{id}/log-entry
```

```json
{
    "raw_text": "Hoy hicimos la actividad de grupo pero no llegamos al cierre porque los chicos se engancharon mucho con el debate."
}
```

Respuesta:
```json
{
    "id": 1,
    "teacher_lesson_plan_id": 42,
    "input_type": "text",
    "raw_text": "Hoy hicimos la actividad de grupo pero...",
    "objective_fulfilled": "partial",
    "activities_completed": false,
    "follow_up_questions": [
        "¿Llegaste a evaluar si entendieron el concepto principal?",
        "¿Cuánto tiempo duró la clase en total?"
    ],
    "created_at": "2026-03-27T18:00:00Z"
}
```

### Listar bitácoras del área (coordinador)

```
GET /areas/{id}/log-entries?from=2026-03-01&to=2026-03-31
```

Respuesta:
```json
[
    {
        "id": 1,
        "lesson_plan": {
            "id": 42,
            "class_number": 3,
            "title": "Ecuaciones de primer grado",
            "subject": "Matemáticas",
            "teacher": "Doc. García"
        },
        "objective_fulfilled": "partial",
        "created_at": "2026-03-27T18:00:00Z"
    }
]
```

## Notas

- El docente solo puede crear/editar bitácoras de sus propios lesson plans
- El coordinador puede leer bitácoras de los docentes de su área
- Al crear la bitácora con texto, se dispara automáticamente el procesamiento estructurado (T-5.6.3)
- Al crear con audio, primero se transcribe (T-5.6.2) y luego se procesa
- El PATCH permite al docente corregir datos extraídos incorrectamente por Alizia
- Los filtros de fecha en el listado del coordinador son opcionales (default: último mes)
