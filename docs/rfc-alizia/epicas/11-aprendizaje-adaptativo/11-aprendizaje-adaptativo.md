# Épica 11: Aprendizaje Adaptativo

> Ajuste automático de contenidos y recomendaciones según el progreso del alumno.

**Estado:** Horizonte
**Fase de implementación:** Por definir

---

## Descripción

Módulo opcional que adapta el contenido y las recomendaciones pedagógicas en función del avance real de cada alumno. Se activa via feature flag (`modules.aprendizaje_adaptativo`) en la configuración de la organización.
