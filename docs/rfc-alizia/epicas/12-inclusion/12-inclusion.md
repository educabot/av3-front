# Épica 12: Inclusión

> Herramientas de accesibilidad y adaptación curricular para alumnos con necesidades especiales.

**Estado:** Horizonte
**Fase de implementación:** Por definir

## Descripción

Módulo opcional que provee herramientas para adaptar planificaciones y contenidos a alumnos con necesidades educativas especiales. Se activa via feature flag (`modules.inclusion`) en la configuración de la organización.
